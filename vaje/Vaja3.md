# 3.1 Business Model Canvas (osredotočeni 4 elementi)
Element	Opis
Customer Segments	Uporabnik: mladi zaposleni (20–35 let), ki uporabljajo mobilno bančništvo, vendar nimajo nadzora nad porabo
Plačnik: uporabnik sam (B2C model)
Value Proposition	Popoln pregled nad financami brez ročnega vnosa + personalizirani AI nasveti za zmanjšanje porabe in boljše finančne odločitve
Revenue Streams	- Freemium (osnovni pregled financ brezplačno)
- Naročnina: 4,99–9,99 €/mesec (napredna analitika, AI priporočila)
- Premium funkcije (npr. finančno načrtovanje, cilji, napovedi)
Cost Structure	- Razvoj & vzdrževanje AI modelov (~50–60 %)
- Cloud infrastruktura (obdelava podatkov, API integracije)
- Marketing (social media, influencerji, performance ads)
- Varnost in skladnost (GDPR, zaščita finančnih podatkov)
Ključno ekonomsko vprašanje:

Če uporabnik zaradi aplikacije prihrani 50–150 € mesečno zaradi boljše kontrole porabe, je naročnina 5–10 €/mesec zelo upravičena (visok perceived value).

# 3.2 Ključne poslovne hipoteze (top 3–5)

1. Uporabniki so pripravljeni plačevati 5–10 €/mesec za aplikacijo, ki jim pomaga zmanjšati nepotrebno porabo.
→ Test: landing page + “fake paywall” (merjenje konverzije)

2. Največja bolečina ni pomanjkanje aplikacij, ampak pomanjkanje avtomatizacije in personalizacije.
→ Test: anketa (50+ uporabnikov) + intervjuji

3. Uporabniki bodo aplikacijo uporabljali vsaj 3× tedensko, če dobijo konkretne in uporabne AI nasvete.
→ Test: MVP + merjenje retention (DAU/WAU)

4. Uporabniki so pripravljeni povezati svoj bančni račun, če je zagotovljena varnost in enostavna uporaba.
→ Test: onboarding flow + % uspešnih povezav

5. Personalizirani “spending alerts” zmanjšajo impulzivno porabo za vsaj 10–20 %.
→ Test: A/B test (z alerti vs brez)

# Naslednji koraki
Validacija hipotez (intervjuji, ankete, landing page)
Razvoj MVP:
povezava z enim bančnim računom
osnovna kategorizacija stroškov
1–2 AI priporočila
Testiranje na manjši skupini uporabnikov (10–20 ljudi)
Merjenje:
uporabe (engagement)
prihranka denarja
zadovoljstva uporabnikov
