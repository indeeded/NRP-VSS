# Prioritization

## Izbrana metoda: MoSCoW

### Zakaj MoSCoW?
Metoda MoSCoW je primerna, ker:
- omogoča jasno razdelitev funkcionalnosti za MVP
- pomaga razlikovati med "must-have" in "nice-to-have"
- je enostavna za uporabo in razumevanje

## Zaključek

MVP se osredotoča na:
- avtomatizacijo (brez ročnega vnosa)
- preglednost financ
- osnovno kontrolo porabe

Napredne funkcije (AI, napovedi) se dodajo kasneje.
